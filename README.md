# Elementary-Status-Widget
## HOW-TO Build
You'll need EFL and Elementary devel packages. 
After installation and `git clone`, follow these steps:
```
# in the project directory
mkdir build
cd build
cmake ..
make
```

## Test using demo app
```
# in the build directory
./status
```
